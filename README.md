# 🚀 SemLimites Prestadores

Plataforma para conectar prestadores de serviços com clientes, com verificação de CNPJ e sistema de reputação.

## 📋 Sobre o Projeto

O SemLimites é um marketplace local de serviços onde:
- **Prestadores** se cadastram, verificam CNPJ e gerenciam sua reputação
- **Clientes** buscam por categoria/cidade, veem avaliações e contratam via WhatsApp
- **Avaliações** são confirmadas pós-serviço, garantindo reputação real

## 🏗️ Arquitetura

- **Frontend**: React PWA (servido pelo GitHub Pages)
- **Backend**: Node.js + Express (Azure App Service)
- **Banco de Dados**: MongoDB (Azure Cosmos DB)
